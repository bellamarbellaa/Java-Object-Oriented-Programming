This repository includes a collection of Java programs developed in my Object-Oriented Programming (OOP) course 
as part of the Information Systems and Applied Computing dual degree program at Sampoerna University and The University of Arizona. 

The projects cover fundamental programming concepts such as hierarchy of Java program (library-package-class),
variable encapsulation (private vs public members), static and constant variables, immporting classes, 
classes relationships, procedural and OOP design, Unified Modular Language (UML) design,
user input handling, output formatting, string representation, and mathematical computations.
